Glossary

  • 404

It’s an error message that indicates that a server (for websites) cannot find the requested content/page from a user. This often happens when accidental hyperlinks are inserted onto a web page or a search query cannot be executed correctly.

  • Alignment

Alignment makes sure that content on a page is aligned and positioned as it is originally planned. Elements/objects can be aligned on a page to the left, right and to the centre. These elements can be aligned also with each other and with other objects.

  • Alt attribute

The alt attribute is usually used in html code especially for representing images on a web page. The “alt” part stands for “alternative”. If an image cannot be displayed on a web page, the alternative text will usually display what the image would have been, what it would have look like.

  • Back-end

The back of a service (software and website) that is usually hidden from users and customers to see. The back-end usually includes functions, information of a structure, applications of websites and software programs.

  • Backlink

Backlinks are also known as trackbacks, pingbacks for blogs. Backlinks are links from third party websites that link back to your own website. These links have a large impact on websites and on the search ranking status (on Google, Bing etc.).

  • Bandwidth

Bandwidth refers to the rate, speed at which data is transferred or the total amount of data that is allowed to be transferred from a web host. The latter is usually capped in certain time periods, usually on a monthly basis. (the more data transferred, the more bandwidth is used above a certain limit, the more charge the owner can expect.)

  • Below the fold

Below the fold is an area on the bottom of a website that can usually be only reached and seen by scrolling down on a web page.

  • Browser

It’s a software application that displays websites on the display. A browser can display and process HTML, scripting languages such as: JavaScript, PHP and CSS. Without a browser, a web page or a whole site would not be possible to be displayed on a device.

  • Cache

Cache stored information of the websites you have recently visited and saves small chunks of information about them. Cache will be used again when a user visits the same website again which will increase the website’s performance as it will use the Cache’s data to restore a website’s content.

  • CSS (Cascading Style Sheets)

CSS is used alongside HTML and scripting languages to enhance a website’s performance and visuals. CSS is responsible for the layout out elements as well as the placement, visual representation of HTML objects on a website.

  • CMS (Content Management System)

CMS is a software that allows users to easily create manage and create content for a website (such as posts/pages and more). These are very popular among bloggers and large corporations.

  • Client-side

Client-side refers to scripts running and being processed on a user’s (client) device/machine. This makes websites load in faster as the process does not take place on the website’s server. This however has the disadvantage of slowing down the performance of a website if the scripts run extensively and severely.

  • Comment

If you take a look at a comment in the HTML, CSS files, you will quickly realise how useful it is to comment in codes. Comments are extremely useful to tell other team mates or just save information about a certain part of a code and describe it in short detail about what it does, what it refences.

  • Compression

Compression refers to files and their sizes. By compressing them, they will lose a large portion of their file sizes and will reduce the storage it takes up on a hard drive/server. This results in faster download/upload speeds. Images and full websites can be compressed by various tools to save storage and make the performance better while not losing any quality of images or content from other files.

  • Cookies

Cookies are small data sets that save a user’s preferences. i.e.: If a user has set an option to be on for a website, the cookie will store that information and load it when the user goes back to the same website.

  • Doctype

Doctype declares a certain page/file’s document type. It is mainly used in HTML at the beginning of starting any code. It defines that a particular page will strictly be based on HTML, therefore a browser should immediately be able to interpret it and display it.

  • Domain

A domain refers to the address of a website. E.g.: www.google.com/ - in this case, Google.com is the overall domain address. All domain addresses are associated with an IP Address that is later on transferred to a server and onto a user’s machine to display the website that is requested.

  • External Style Sheet

It’s a CSS document that is written separately and outside of an HTML document. The external CSS sheet is connected via a line of HTML code inside the HTML document in the head tag.

  • Favicon

It’s a small icon that is associated with a website’s brand image and logo. This is usually displayed on the title bar of a browser and on bookmarks.

  • Fork

The term “fork” is referred to copying and using someone else’s work. When someone else’s source code I copied, an individual or a team will usually take it apart and develop, improve on it. This is cost and time effective as it can save a lot in the R&D department as well a save time while doing so.

  • Front-end

The “Front-end” term refers to the opposite of “back-end”. Front-end, the term itself is mainly used to describe what a user sees on the website as they are often not allowed to see how a website is structured (back-end). The front-end displays usually the design, function elements on a website that can be publicly seen.

  • FTP (File Transfer Protocol)

It’s a standard network protocol that is used to transfer files from a machine to a server via the internet.

  • GUI (Graphical User Interface)

The Graphical User Interface can usually be used with output devices such as mouse, keyboard and touch, pen nowadays. The GUI is a visual representation of a web application that offers content, functions to interact with.

  • Hex Code

The “hex” code is mainly used to display colours. The Hex language includes numbers from 0-9 and letters from A-F. Each Hex colour code includes three letters and three numbers to pick one colour out of millions.

  • HTML (Hypertext Mark-up Language)

HTML is the language that is used to structure and build a web page. HTML code can display images, videos, audio and can refer to CSS and script files to enhance a certain website’s functionality and make it look visually appearing.

  • Hyperlink

Hyperlinks are links. It’s a link that connect pages and other contents on a website together. Without hyperlinks, users wouldn’t be able to navigate from one page or website to another.

  • iFrame

It’s a frame that requires a specified amount of space by the user. These frames are most used to display videos and images in a certain area.

  • JavaScript

It’s a scripting language that is used to make websites look visually better as well as to enhance its functions with useful and easy to use options and settings.

  • JPEG

It’s perhaps the most popular file format ever. It uses both lossy and lossless compressions with marginally low loss of quality. It’s widely accepted to be used for photographs, images on the internet as well as for paintings even.

  • Landing Page

The landing page refers to the page a visitor usually first visits and sees on a website. Landing pages are often, and include marketing campaigns and promotions from advertising companies or from the website’s owners/companies products.

  • Liquid Layout

It is referred as “liquid” due to the fact that a page is based on percentages of a browser’s window size. The layout and the whole website will change and be affected by the browser’s size/screen estate on which it is displayed on. This ensures that the website is optimised and shows up properly for both on small and on large screen sizes.

  • Meta Data

Meta data is essentially text, and could be other data formats that describe a certain data in short details.

  • Open Source

Open source refers to the source code of a program, website being publicly shown and making it available for use and view. Open source code is usually very cost effective and can be developed in teams and individually by anyone and anywhere.

  • Pixel (px)

Image element. It is used in all industries. It’s a unit of a point on a device. It is used to measure dimensions of an image and other certain aspects of content such as websites. Dimensions are display as: 1080px by 1920px.

  • PNG

It’s an image file format that is widely used for web based images. PNG is mainly used when an image has a large area of the same colour or when an image has a transparent background in it. (widely used for logos)

  • Registrar

It is usually a company from which, through a person has registered and purchased a domain name. An example for this would be: GoDaddy.

  • Resolution

It’s the physical number of pixels that can be displayed on a certain device’s screen. The resolution however does not refer to how many pixels are placed on the screen. The resolution is usually displayed as: 1920 x 1080.

  • Script

It’s usually refers to the code that is placed in the HTML that makes a website feature rich or in other words: interactive and dynamic. There are various scripting languages used for website, the most popular one being JavaScript.

  • Server-side

Server-side refers to scripts running on a server instead of running of a user’s machine/device. Server-side usually offers a more stable and faster process of handling things on websites, however this comes at a cost of reloading. Page usually must be reloaded in order to see server side changes appear on a user’s screen.

  • Template

A template is a file that is used to offer a consistent design and structure for a whole website.

  • URL (Uniform Resource Locator)

It’s an address that is easy to remember and sends a user to a specified web address where a website is located at.

  • Usability

Usability refers to “how easy” is a website to use and operate. It refers to how a website can be used, how it’s functions work and how easy and convenient it’s objects are in their structured positions.

  • User Interface (UI)

User experience defines the control and the interaction between a user and a computer with a graphical interface. There are different types of interfaces such as: graphical, auditory and textual ones.

  • User Experience (UX)

The User Experience (UX) is an element of design that references a user’s interaction with a certain product/service or a graphically shown interface.

  • Web Page

A page is essentially a document that can be represented on a user’s device by requesting it over the World Wide Web (www). The page file/document will be transferred over from the website’s server.

  • Web Server

A server that contains information and overall the whole website’s content on a server. This server transfers it’s files to the internet and to other users to display a certain website on their devices.

  • Web Standards

Web Standards are the standards that are set by the World Wide Web Consortium to standardise the structure and the overall build quality of the websites. They maintain consistency and help companies make more suitable, easier websites in the future. These standards also make sure that all websites that follow the standards will show up in browsers as they are originally intended to show up.

  • Wireframe

It’s a web design tool that is used to give a visual representation of a proposed function, structure, design and layout of a future website.

  • www (World Wide Web)

The World Wide Web is a subnet that requires HTTP/HTTPS protocols to send web pages and documents over the Internet.

  • Zip file

It’s a popular file compression format. It uses the .ZIP extension file format, hence the name of calling it “Zip file”. ZIP files usually contain web and other files that provides a faster download speed over the traditional folders. Once a ZIP file is downloaded by a person, it can be UNZIPPED to open up the documents and files and view, edit them.
